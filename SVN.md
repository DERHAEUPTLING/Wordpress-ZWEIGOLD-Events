# svn workflow

https://plugins.svn.wordpress.org/regiondo-widgets

https://developer.wordpress.org/plugins/wordpress-org/how-to-use-subversion/


**update local copy**
svn up

**add folders**
svn add trunk/*
svn add assets/*

**delete**
svn delete trunk/delete-me

**commit**
svn ci -m 'message'

**git status**
svn stat
